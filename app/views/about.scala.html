What Blogerei Is About

I'm a Ruby Developer who's interested in Scala programming language. Of course it's never only the programming language itself but also an enormous amount of related tools that are available. So, by saying Scala I assume the Play Framework, Slick Database, Akka, SBT and many more. I won't try to write about everything in this blog but rather stick to the language and use the power it provides. Each post has pretty much the same structure:

  • Description of a certain problem that needs to be solved
  • One or more solutions that I've come up with
  • Specs that the code should pass
  • Comments (better solutions?) from the readers

By the way, this blog was also developed with Scala and Play Framework. You can contribute or just get familiar with the source code on github.
